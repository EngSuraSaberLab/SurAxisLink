# SurAxisLink

Static portfolio site prepared for GitHub Pages.

## Live Structure

- `docs/` contains the full published website.
- `docs/index.html` is the English homepage.
- `docs/ar/index.html` is the Arabic homepage.
- `docs/assets/` contains all CSS, JS, images, and vendor files used by the site.

## Editing

- Edit page content directly inside `docs/`.
- Edit styling inside `docs/assets/css/`.
- Edit interaction behavior inside `docs/assets/js/main.js`.

## GitHub Pages

Use GitHub Pages with:

- Branch: `main`
- Folder: `/docs`
